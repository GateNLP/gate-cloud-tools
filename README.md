# GATE Cloud tools

This is a repository of useful tools for use with the [GATE Cloud](https://cloud.gate.ac.uk) platform.

## Available tools

- [CSV Processor](./process-csv/): tool to process a CSV file of text documents through a GATE Cloud service

Other tools may be added in future.
